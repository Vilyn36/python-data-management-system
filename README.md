# Python Data Management System

This project implements a simple data management system using Python, focusing on business logic, data validation, and data persistence.

## Overview
The system simulates the registration and management of structured records.
It emphasizes clean logic, validation rules, and organized data handling.

## Features
- Data validation and normalization
- CSV-based persistence
- Optional Excel (.xlsx) integration
- Modular and reusable functions
- Clear separation between logic and storage

## Technologies Used
- Python
- CSV
- openpyxl

## Purpose
This project was created to demonstrate logical reasoning, data handling, and structured Python code in a real-world scenario.
